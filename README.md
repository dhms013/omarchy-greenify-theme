# Installation Theme

To install this theme, simply use the omarchy-theme-install command:

```
omarchy-theme-install https://github.com/dhms013/omarchy-greenify-theme
```

For other configs, please check this [my dotfiles](https://github.com/dhms013/dhmsDots/tree/main)
